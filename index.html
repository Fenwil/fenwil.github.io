MercadoLibre Mas Vendidos

The problem: MercadoLibre website doesn't offer a way to order the products descendantly from the most selled to the less selled.

The solution: a Google Chrome extension that takes all items, orders it and then insert it back to the DOM.

The numbers: more than 300 users and 9 ratings with 5 stars.

The stack: ECMAScript6, Gulp, Babel, Prettier and XO.

Prettier-rator

The problem: each Prettier plugin for each code editor like Sublime Text, Atom, Visual Studio Code has a different syntax and file extension.

The solution: a CLI that takes input for each setting in Prettier and creates a file for each plugin so you can share it across your team.

The numbers: more than 700 downloads with a peak of 345 downloads in a month.

The stack: readline-sync, commander, jsonfile, chalk.

Postal Code

The problem: if you need to find the postal code of your direction you have to fill ugly forms inserting lots of information.

The solution: a simple web app that takes your current coordinates and shows the postal code for that particular coordinates.

The stack: React, react-spinkit, OSMP Nomatim API.
